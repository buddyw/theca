Updated!

Theca is once again a maintained project! It has been updated to compile on all current versions of rustc, linted with clippy, and regularly building on travis. I have updated the version to 1.0.0, and published it to crates.io to allow for installation via cargo install.

I am welcoming bugs/features/whatever, so please cargo install theca, start using it, and file bugs at https://github.com/pwoolcoc/theca!
